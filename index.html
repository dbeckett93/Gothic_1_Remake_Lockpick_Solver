⚙ Gothic 1 Remake — Lockpick Solver

The chest lock is a set of coupled plates. Moving one plate can nudge others (sometimes in the opposite direction). Tell the solver how your lock is wired and where the plates sit now, and it finds the shortest sequence of moves that lines every plate up on pin 4.

Edge behaviour
 

Walls is how the real game works: a pin can't leave holes 1–7, so any push (including the coupled knock-on) that would shove a pin off an edge is refused. Wrap is an experimental mode where pins loop around — only use it if your lock really behaves that way.  ·  Optimise for: Fewest moves = least total nudges; Fewest switches = least times you re-select a plate (the slow in-game action), so each plate's nudges are grouped together.

2 · The lock & current positions

Click a slot to set where a plate sits right now. Use the ◀ ▶ buttons to test a real move — couplings apply, so you can check the wiring matches your game. Plate is green when it's on pin 4.

3 · How the plates are wired (couplings)

Each row says what happens to every plate when you push that plate one notch right. Click a cell to cycle → (also right), ← (goes left), · (unaffected). Pushing left just does the opposite. How to find this in-game: from a calm lock, nudge one plate a single notch and watch which others move, and which way. The brass-edged diagonal is the plate you're pushing (normally →).
