⚙ Gothic 1 Remake — Lockpick Solver

The chest lock is a set of coupled plates. Moving one plate can nudge others (sometimes in the opposite direction). Tell the solver how your lock is wired and where the plates sit now, and it finds the shortest sequence of moves that lines every plate up on pin 4.

1 · Lock setup

Plates
 

A pin can't leave holes 1–7, so any push — including the coupled knock-on movement of a linked plate — that would shove a pin off an edge is refused, exactly like the game. The solver finds the fewest total moves and never asks for an illegal push.

2 · The lock & current positions

Click a slot to set where a plate sits right now. Use the ◀ ▶ buttons to test a real move — couplings apply, so you can check the wiring matches your game. Plate is green when it's on pin 4.

Plate order

Flip the stack so it matches what you see in‑game, then read the plates off in that order. P1 is only a label — it can be any physical plate; the solver just needs you to stay consistent. The solution's plate numbers follow this same top‑to‑bottom order.

3 · How the plates are wired (couplings)

The grid starts empty — fill in only what your lock actually does. Each row is one plate you push one notch right; click a cell to set how another plate reacts: → moves the same way, ← moves the opposite way, · no effect. Pushing left just mirrors it. The shaded • on the diagonal is the plate itself — it always moves when you push it, so you never set it. To map a lock: tap one plate a single notch in-game and watch which others move, and which way.

Here, pushing P1 right also shoves P2 right; pushing P2 right pulls P3 the other way (inverted). Everything else is blank. Push left and the same links run in reverse.
